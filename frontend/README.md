# Todo List App

A full-stack todo list application built with React, Express, and TypeScript.

## Features
- Add, complete, and delete todos
- Filter by All / Active / Completed
- Persistent storage (JSON file on the backend)

## Tech Stack
- **Frontend:** React, TypeScript, Vite
- **Backend:** Node.js, Express, TypeScript
- **Storage:** JSON file

## Running Locally

### Backend
\`\`\`bash
cd backend
npm install
npm run dev
\`\`\`
Runs on http://localhost:3001

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
Runs on http://localhost:5173
